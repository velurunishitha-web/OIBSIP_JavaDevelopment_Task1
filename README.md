# Online Reservation System

## Objective

The objective of this project is to build a simple Online Reservation System using Java. The system helps users reserve and cancel train tickets easily through a user-friendly interface.

## Features

* Login using Login ID and Password
* Reservation Form
* Passenger Details Entry
* Automatic Train Name Display
* Class Type Selection
* Journey Date and Destination Details
* PNR Number Generation
* Ticket Cancellation using PNR Number
* Logout Option

## Tools Used

* Java
* VS Code
* JDK

## Steps Performed

1. Created login page using Login ID and Password.
2. Added reservation form for entering passenger details.
3. Implemented automatic train name display using train number.
4. Added journey details like class type, source and destination.
5. Generated PNR number for ticket booking.
6. Implemented cancellation form using PNR number.
7. Added logout functionality.

## Outcome

The project was successfully created and tested in Java. It provides an easy and efficient way for users to reserve and cancel train tickets through a simple console-based system.
